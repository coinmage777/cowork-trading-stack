# 00. 시작하기 전에

이 가이드를 따라가기 전에 먼저 분명히 해두고 싶은 것이 몇 가지 있습니다.

## 이 가이드는 무엇이고 무엇이 아닌가

**이 가이드는** 직접 운영해온 크립토 트레이딩 자동화 시스템에 대한 작업 노트입니다. 페어 트레이딩 봇, [Polymarket](https://polymarket.com/?ref=coinmage) 자동매매, 김프 차익, 거래소 간 아비트라지, 옵시디언 기반 리서치 파이프라인, AI 에이전트 통합까지 — 거의 매일 사용하는 도구들입니다.

**이 가이드는 아닙니다**:
- 부자 되는 법 가이드. 이 글을 읽고 누군가 갑자기 큰돈을 벌었다는 이야기를 듣는 일은 절대 없을 것입니다.
- "확정 수익" 전략 모음. 모든 전략은 망할 수 있고, 운영 전략 중 일부도 며칠씩 손실난 사례가 많습니다.
- 완벽한 코드. 운영해온 봇은 깨지고 패치하고 다시 깨집니다. 그 과정 자체가 일입니다.

투자 조언이 아닙니다. 운영 방식에 대한 기록일 뿐입니다.

## 전제 조건

### 기술 스택
- **Python 3.10+** — 봇 대부분이 비동기 Python
- **Node.js** — 일부 SDK가 JS 전용
- **Git** — 버전 관리 없이 봇을 운영하면 한 번은 망합니다
- **VSCode 또는 Cursor** — AI 통합 에디터 권장
- **터미널** — bash/zsh/PowerShell 어느 것이든 익숙해야 합니다

### 인프라
- **로컬 머신** — 개발용, MacBook 또는 Windows
- **VPS** — 운영용, Contabo / Hetzner / DigitalOcean / AWS 중 어디든 OK
- **Google Drive** 또는 동등한 클라우드 — 콘텐츠 백업
- **Telegram** — 알림 + 원격 제어

### 자본
- 처음에는 적게. 정말 적게. 봇이 안정적으로 일주일 굴러가기 전에는 큰돈을 넣지 마세요.
- 처음에는 거래소당 $50–100 수준으로 시작하는 것이 일반적입니다.
- 자본 규모는 전략의 안정성이 검증된 다음에야 늘리는 것입니다.

### 시간
- 처음 셋업: 주말 두 번 정도
- 첫 봇이 안정 운영까지: 2–4주
- 매일 모니터링: 30분~1시간
- 큰 변경 / 디버깅: 그때그때 다름

## 이 가이드의 작성 방식

각 장은 독립적으로 읽을 수 있게 작성되었습니다. 다만 순서대로 보면 빌드업이 자연스럽습니다:
1. 자동화 마인드셋 잡기 (1장)
2. AI 도구 셋업 (2–3장)
3. 지식 인프라 구축 (4장)
4. 가벼운 자동화부터 (5장: 볼륨 파밍)
5. 본격 트레이딩 봇 (6–10장)
6. 운영 / 인프라 / 거래소 셋업 (11–12장)
7. 로드맵 + 참고 (13–15장)

## 보안 원칙 — 진짜로

- 프라이빗 키 / API 시크릿 / 패스프레이즈는 **절대로** 코드에 직접 넣지 마세요
- `.env` 파일은 git에 커밋하지 마세요 (`.gitignore`에 등록)
- API 키는 트레이딩 권한만 주고, **출금 권한은 절대 X**
- IP 화이트리스트가 가능하면 무조건 켜세요
- 멀티시그 / 하드웨어 지갑은 큰 자금에는 필수입니다

이 가이드의 모든 코드 예시는 민감 정보가 placeholder로 처리되어 있습니다. 자신의 키를 그 자리에 채워 넣을 때 한 번 더 확인하시기 바랍니다.

## 이 자료의 성격

이 자료는 출판 시점의 스냅샷입니다. 전략과 도구는 계속 바뀌기 때문에 본문 중 일부는 이미 구버전일 수 있습니다. 고정된 매뉴얼이라기보다 작동 중인 프레임워크로 읽으시면 됩니다.

좋은 알파는 글로 적는 순간 알파가 아니게 됩니다. 이 자료의 목적은 "지금 당장 따라하면 돈이 벌리는 비법"이 아니라, "이런 방식으로 일하면 자기만의 접근을 만들 수 있다"는 프레임워크 제시에 가깝습니다.
