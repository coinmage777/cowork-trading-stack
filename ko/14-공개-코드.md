# 14. 공개 코드

이 장에서는 트레이딩 자동화 스택을 구축할 때 참고할 만한 오픈소스 자원, SDK, 도구들을 정리합니다.

## 학습용 오픈소스 봇

### Freqtrade
- https://github.com/freqtrade/freqtrade
- Python 기반 트레이딩 봇 프레임워크
- Backtesting + Paper trading + Live 다 지원
- 다양한 CEX 지원 (ccxt 기반)
- 시작용으로 가장 추천

처음 페어 트레이딩을 익힐 때 freqtrade의 백테스팅 모듈을 참고했습니다.

### Hummingbot
- https://github.com/hummingbot/hummingbot
- 마켓메이킹 / 차익거래 특화
- 알고 트레이딩 입문 강력 추천

### Jesse
- https://github.com/jesse-ai/jesse
- Python, 깔끔한 API
- 백테스팅 강력함

### CCXT
- https://github.com/ccxt/ccxt
- 100+ 거래소 통합 라이브러리
- CEX 쪽은 ccxt에 의존합니다

## 학습 순서 추천

1. **freqtrade** 로 시작해서 백테스트 / 페이퍼 / 라이브 흐름 익힘
2. **ccxt** 로 다양한 거래소 API 친숙
3. **단순 시그널 백테스트** 직접 코딩 (z-score / RSI / 모멘텀)
4. **Hyperliquid SDK** 로 첫 DEX 봇 (페이퍼)
5. **본격 봇 빌드** — 환경에 맞춰 from scratch

각 단계마다 코드 한 줄도 안 짤 자신 있는 사람은 잘못된 방향입니다. AI가 90% 짜주는 시대지만, 그 코드를 이해 못하면 운영 사고 시 대응을 못 합니다.
